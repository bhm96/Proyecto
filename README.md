Mi primer proyecto
==========

Este proyecto permite ingresar un nombre mediante consola para después lanzar un mensaje de bienvenida con el nombre ingresado anteriormente.

Utilidad
--------------------

Es útil si se dar una bienvenida se trata, ya que mediante el nombre ingresado por consola logra identificarte al lanzar el mensaje final.
